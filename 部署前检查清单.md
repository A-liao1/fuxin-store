# 部署前检查清单

## 数据保护机制确认 ✅

### 1. 数据库备份（3层保护）

#### 第一层：部署前备份
- 位置：`/root/backups/database.db.YYYYMMDD_HHMMSS.bak`
- 时机：部署开始时
- 保留：最近10个备份

#### 第二层：临时保护
- 位置：`/tmp/database.db.keep`
- 时机：删除旧部署前
- 用途：确保数据库不会丢失

#### 第三层：部署历史备份
- 位置：`/root/fuxin.backup.YYYYMMDD_HHMMSS/`
- 内容：完整的部署目录（包含数据库）
- 保留：最近5个

### 2. 数据恢复流程

```
1. 停止服务
2. 备份当前数据库 → /root/backups/
3. 复制数据库到临时位置 → /tmp/database.db.keep
4. 备份整个部署目录 → /root/fuxin.backup.*
5. 删除旧部署目录
6. 解压新版本
7. 恢复数据库 ← /tmp/database.db.keep
8. 启动服务
```

### 3. 数据不会丢失的原因

- ✅ 部署前先备份到 `/root/backups/`
- ✅ 删除前复制到 `/tmp/database.db.keep`
- ✅ 解压后立即恢复数据库
- ✅ 整个部署目录也有备份
- ✅ 即使失败，也有多个备份可恢复
