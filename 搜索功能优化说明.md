# 搜索功能优化说明

## 优化内容

将采购入库、销售出库和结算管理的所有搜索功能从前端过滤改为后端过滤，提升性能和用户体验。

### 1. 采购入库管理
- 原来：下拉选择供应商ID（前端过滤）
- 现在：输入框模糊搜索供应商名称（后端过滤）

### 2. 销售出库管理
- 原来：下拉选择客户ID（前端过滤）
- 现在：输入框模糊搜索客户名称（后端过滤）

### 3. 结算管理

#### 采购应付账单
- 原来：下拉选择供应商ID（前端过滤）
- 现在：输入框模糊搜索供应商名称（后端过滤）

#### 销售应收账款
- 原来：下拉选择客户ID（前端过滤）
- 现在：输入框模糊搜索客户名称（后端过滤）

#### 结算记录
- 原来：下拉选择供应商/客户ID（前端过滤）
- 现在：输入框模糊搜索供应商/客户名称（后端过滤）
- 单据类型和支付方式保持下拉选择（固定选项）

  if (supplierName) {

## 使用方法

### 采购入库搜索
1. 在"供应商名称"输入框输入关键词（如"茅台"）
2. 点击"搜索"按钮
3. 系统会在后端查询所有供应商名称包含"茅台"的采购单

### 销售出库搜索
1. 在"客户名称"输入框输入关键词（如"张三"）
2. 点击"搜索"按钮
3. 系统会在后端查询所有客户名称包含"张三"的销售单

### 结算管理搜索

#### 采购应付账单
1. 在"供应商名称"输入框输入关键词
2. 可配合日期范围筛选
3. 点击"搜索"按钮

#### 销售应收账款
1. 在"客户名称"输入框输入关键词
2. 可配合日期范围筛选
3. 点击"搜索"按钮

#### 结算记录
1. 可选择"单据类型"（采购单/销售单）
2. 在"供应商名称"输入框输入关键词（适用于采购单）
3. 在"客户名称"输入框输入关键词（适用于销售单）
4. 可选择"支付方式"
5. 可配合日期范围筛选
6. 点击"搜索"按钮

## 优势

1. 性能提升：后端过滤减少数据传输量
2. 用户体验：模糊搜索比下拉选择更灵活
3. 可扩展性：支持大量供应商/客户时不会影响性能
4. 搜索精度：支持部分匹配，更容易找到目标
5. 统一体验：所有搜索功能保持一致的交互方式
