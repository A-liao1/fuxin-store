## 如果出现问题

### 问题1：数据丢失
**立即执行：**
```bash
./emergency-restore.sh
```
选择最新的备份文件恢复

### 问题2：环境变量未设置
**检查：**
```bash
ssh root@104.223.59.170 'pm2 show fuxin'
```

**修复：**
```bash
ssh root@104.223.59.170 'pm2 delete fuxin && cd /root/fuxin && NODE_ENV=production pm2 start dist/index.js --name fuxin --env production'
```

### 问题3：服务无法启动
**查看日志：**
```bash
ssh root@104.223.59.170 'pm2 logs fuxin --lines 50'
```

**重启服务：**
```bash
ssh root@104.223.59.170 'pm2 restart fuxin'
```

## 回滚方案

如果部署失败，执行以下步骤：

### 1. 停止新服务
```bash
ssh root@104.223.59.170 'pm2 stop fuxin'
```

### 2. 恢复备份
```bash
ssh root@104.223.59.170 'cp /root/backups/database.db.20260222_203831.bak /root/fuxin/database.db'
```

### 3. 恢复旧版本代码
```bash
ssh root@104.223.59.170 'cd /root && rm -rf fuxin && cp -r fuxin.backup.YYYYMMDD_HHMMSS fuxin'
```

### 4. 重启服务
```bash
ssh root@104.223.59.170 'cd /root/fuxin && pm2 start fuxin'
```
